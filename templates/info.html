Information

Memory Kards is a flashcard application where the user can create cards to help memorize concepts of anything interesting for the user. It was developed as the final project for CS50x 2023.
Each card contains a question and an answer to it, and optionally a tag name, which can be used as filter when selecting which cards to read in the train section. Questions must be unique, while answers can be the same for multiple cards.
It's also possible to delete and edit existing cards if the user wants to.
